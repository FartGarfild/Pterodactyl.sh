Это скрипт для быстрой установки панели pterodactyl на CentOS 7.
В скрипте нет доступа к почте. Так как его не делал. Этот скрипт устанавливает базовую рабочую версию панели.
Спрашивает домен 2 раза. (Первый раз нужен для конфигураций apache а второй для конфигурации панели).
Так же сначала скрипт попросит создать пароль для бд (Если просто нажать enter ничего не написав то будет указан пароль 6482).
Этот же пароль нужно будет вписать позже при настройке подключения к БД.
Один раз будет спрашивать Egg author Email - тут указываем любую почту.
Application URL обязательно нужно писать в виде http://sie.com (просто домен оно не примет, хотя оно должно создавать само конфигурации apache, оно их не создаёт, по этому скрипт делает это сам.)
На моменте TimeZone можно просто нажать Enter.

Для правильной установки нужно выбрать.
Cache Driver - File
Session Driver -database
Queue Driver - database

При настройке базы данных нужно вводить только пароль. Так как скрипт создаёт стандартную базу к которой ранее создавали пароль.
Но Вы можете указать свою Базу данных на другом сервере.
Если неправильно подключили БД, нужно войти в каталог /var/www/pterodactyl/ и ввести команду
php artisan p:environment:database (Она отвечает за конфигурацию для подключения к базе данных). Либо перезапустить скрипт.
